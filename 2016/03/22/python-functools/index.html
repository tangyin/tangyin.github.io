from functools import wraps

wraps 一般用在装饰器那里，装饰器的本质是用一个方法替换掉传入的那个方法。简单点说我传入的是方法A，然后得到了方法B。这个时候就会有一个问题，我方法A的一些属性在传递过程中丢了，比如dicstring。wraps就是来解决这个问题的，他帮我们把方法A的属性一起传了过去。下面写几个例子吧

f这个方法其实已经变成了with_logging了，原来的属性全部丢失

如果加上了wraps这个装饰器，那么f的属性就被传递了过去，就算我在with_logging也谢了docstring,也会被覆盖掉

from functools import update_wrapper

wraps是update_wrapper这个方法的装饰器实现

from functools import partial

partial函数，它可以重新绑定函数的可选参数，生成一个callable的partial对象
